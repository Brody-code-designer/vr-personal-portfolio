Persona Katana

Katana Grace

Profession

Katana is the lead developer for a gaming studio. She leads a team of developers. The team ensures that updates to various games are released periodically. As team leader she is part of the interviewing process when they are hiring new employees.

Technology

  • Alienware M17 R3 Gaming Laptop; i9 process; Windows 10; NVIDIA® GeForce® RTX 2080 SUPER™ 8GB GDDR6 - Max Performance; 32GB DDR4 RAM; 1TB Storage
  • Oculus Quest 2; uses Virtual Desktop
  • Desktop Browser Mozilla Firefox and Google Chrome. Oculus Browser when not using Virtual Desktop.
  • Samsung Galaxy A21; 720 X 1600 pixels, 20:9 ratio (~270 ppi density); OS Android 10, One UI 2.0; accelerometer
  • Tech Ability: Katana is very tech savy. She spends a lot of time browsing the internet and exploring interesting websites. When a new site is found she understands when it is unique and appreciates the time it must have taken to build. In addition, she enjoys looking at sites on her VR headset. She is very tech savy with her VR headset and has the latest hookup with Virtual Desktop.

Attitudes & Needs

  • Katana is looking at web developer portfolios to determine who would be a good fit to join her team.

Frustrations

  • Very slow websites
  • Websites that are not responsive
  • Bugs and glitches
  • Unreadable text sizes and text colors

Role

Casual User

User Story

As a casual user, I want to see a web developer portfolio.

Use Case

Katana visits my portfolio site and explores the site with her VR headset.

Interaction Flow

  • Katana visits the site
  • The server displays the Home Page
  • Katana moves her mouse cursor
  • The server moves the 3d canvas
  • Katana puts on Oculus with Virtual Desktop and clicks start VR Button
  • Server opens VR app
  • Katana looks around with headset on
  • Server responds with controllers that moves 3d canvas
  • Katana clicks a icon
  • Server opens up more info based on the clicked icon
